# Student-Performance-DataWrangling

A practical data wrangling and preprocessing project built in R using the “Students Performance in Exams” dataset from Kaggle.

This project focuses on understanding data structures, importing datasets, inspecting variables, handling categorical and numeric data types, performing subsetting operations, converting data structures, and creating custom data frames for analysis.

---

## Project Overview

The objective of this assessment is to explore and preprocess a real-world dataset using R programming.

The project demonstrates:
- Importing CSV data into R
- Understanding dataset structures
- Handling categorical and numeric variables
- Factor conversion and level inspection
- Data subsetting and matrix conversion
- Creating custom data frames from scratch
- Basic exploratory data inspection
